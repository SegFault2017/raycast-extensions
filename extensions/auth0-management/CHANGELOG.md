# Auth0 Management Changelog

## [Initial Release] - {PR_MERGE_DATE}

### Features

- **Search Users** — search Auth0 users by name, email, or user ID with wildcard matching
- **Create User** — create new users with email, name, password, and connection picker
- **User Detail** — view full profile, identities, roles, and organization memberships
- **Sessions & Grants** — view active sessions, OAuth2 grants, revoke sessions/grants
- **View Blocked Users** — list blocked users and unblock them
- **User Logs** — view logs filtered to a specific user with date range filtering
- **Switch Tenant** — add, edit, delete, and switch between Auth0 tenants (dev, staging, production)
- **View Organizations** — browse organizations, view members, assign users, create new organizations
- **View Logs** — browse tenant logs with text search, date pickers, time presets, and custom ranges
- **View Apps** — browse Auth0 applications and their configuration
- **View APIs** — browse resource servers, view scopes, add/edit/delete permissions
- **View Databases** — browse database connections and create new ones with authentication method, custom database, sign-up, and domain-level options
- **Multi-tenant support** — manage multiple tenants with color-coded environment badges and per-command tenant dropdown
- **Deep links** — open any user, organization, app, API, or log entry directly in the Auth0 Dashboard
